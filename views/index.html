URL Shortener Microservice

A solution to the URL Shortener MS challenge.

Features

  • Pass a URL as a parameter and return a shortened URL in the JSON response
  • For an invalid URL that doesn't follow the 'http://www.example.com' format, the JSON response will contain an error instead.
  • When I visit the shortened URL, it will redirect me to my original link

Example Usage

Create via FORM 

        curl -d "url=https%3A%2F%2Ffreecodecamp.org" https://jy-fcc-url-short.glitch.me/api/shorten
        {"original":"https://freecodecamp.org","short":"412557052"}

Create via JSON 

        curl -d '{"url": "https://freecodecamp.org"}' -H "Content-Type: application/json"  https://jy-fcc-url-short.glitch.me/api/shorten
        {"original":"https://freecodecamp.org","short":"412557052"}

Get Shortented 

        curl -v https://jy-fcc-url-short.glitch.me/412557052
        HTTP/1.1 302 Found
        Date: Sat, 03 Feb 2018 18:20:21 GMT
        Content-Type: text/plain; charset=utf-8
        Content-Length: 46
        Connection: keep-alive
        x-powered-by: Express
        location: https://freecodecamp.org
        vary: Accept

        Found. Redirecting to https://freecodecamp.org

Get Object from API 

        curl -H "Content-Type: application/json" https://jy-fcc-url-short.glitch.me/api/412557052
        {"original":"https://freecodecamp.org","short":"412557052"}%

Invalid URL sent 

        curl -d "url=nopenopenope.dsaf" https://jy-fcc-url-short.glitch.me/api/shorten
        {"error":"Invalid url"}

Short link Not found 

        curl https://jy-fcc-url-short.glitch.me/apple
        {"error":"Not found"}
